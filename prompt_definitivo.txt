## DESIGN E ESTILO
Aplique os princípios das skills instaladas: `gpt-taste`, `antigravity-design-expert`, `landing-page-design`.

- **Paleta de cores:** Escolha uma paleta sofisticada que combine com perfil técnico/IA. Sugestões:
  - Opção A: Azul marinho profundo (#0a192f) + dourado (#ffd700) + cinza claro (#f8f9fa)
  - Opção B: Verde-escuro (#064e3b) + branco (#ffffff) + cinza (#e5e7eb)
  - Opção C: Roxo profundo (#4c1d95) + branco (#ffffff) + cinza claro (#f3f4f6)
  - Escolha a que melhor transmita sofisticação técnica e profissionalismo

- **Tipografia:** Fonte sans-serif moderna. Use `Inter` via Google Fonts CDN (única exceção permitida) ou `system-ui` stack
- **Tamanhos:** Hierarchy clara — H1 grande no hero, H2 para seções, H3 para subseções
- **Espaçamento:** Generoso padding entre seções (80–120px desktop, 48–64px mobile)
- **Bordas arredondadas:** 8–12px nos cards e botões
- **Sombras:** Sutis (`box-shadow: 0 4px 6px -1px rgba(0,0,0,0.1)`)
- **Animações de scroll reveal:** IntersectionObserver para revelar elementos ao entrar na viewport (fade-in + slide-up)
- **Totalmente responsivo:** Mobile-first, breakpoints em 768px e 1024px

## SEO E PERFORMANCE
- `<title>`: "Fabiano de Oliveira Fernandes — Analista de Sistemas Sênior | Product Owner | AI Developer"
- Meta description com resumo profissional (máx 160 caracteres)
- Open Graph tags: `og:title`, `og:description`, `og:type=website`, `og:url`
- Twitter Card tags
- Structured data JSON-LD:
  - `@type: Person` para dados pessoais
  - `@type: CreativeWork` para cada projeto GitHub
- Favicon (link para favicon genérico ou gerar SVG inline)
- Semantic HTML5: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- Acessibilidade:
  - `alt` tags em todas as imagens
  - `aria-label` em links e botões
  - Contraste mínimo 4.5:1
  - Foco visível em elementos interativos
- Performance:
  - CSS e JS em arquivos separados (não inline)
  - Imagens otimizadas
  - Sem render-blocking resources
