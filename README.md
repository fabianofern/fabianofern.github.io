# Portfólio Profissional - Fabiano de Oliveira Fernandes

Este é o código-fonte da minha landing page de currículo profissional, desenvolvida puramente com HTML5, CSS3 e JavaScript Vanilla. Nenhuma dependência externa, framework ou biblioteca (com exceção de ícones via CDN) foi utilizada, garantindo altíssima performance, acessibilidade e SEO otimizado.

## 🚀 Como Visualizar Localmente

O projeto não requer nenhum servidor ou passo de build. Para visualizar:

1. Clone o repositório ou baixe os arquivos.
2. Dê um duplo-clique no arquivo `index.html` para abri-lo no seu navegador padrão.
3. *Opcional:* Se você utiliza o VS Code, pode usar a extensão "Live Server" para desenvolvimento em tempo real.

## 🌐 Como Fazer o Deploy no GitHub Pages

Este projeto está pronto para ser hospedado gratuitamente no GitHub Pages. Siga os passos abaixo:

1. **Crie um Repositório:** 
   - Vá ao GitHub e crie um novo repositório chamado `seunomedeusuario.github.io` (substitua pelo seu usuário real).
   - Se já tiver um repositório para outro fim, você pode nomeá-lo como `curriculo` (o site ficará acessível em `seunomedeusuario.github.io/curriculo`).

2. **Envie os Arquivos:**
   - Faça o commit e push de todos os arquivos deste projeto (`index.html`, pasta `css/`, pasta `js/`, pasta `assets/`, etc.) para a branch `main` ou `master` do repositório recém-criado.

3. **Ative o GitHub Pages:**
   - No repositório do GitHub, vá em **Settings** > **Pages** (no menu lateral esquerdo).
   - Na seção "Build and deployment", certifique-se de que a fonte ("Source") está definida como **Deploy from a branch**.
   - Em "Branch", selecione `main` (ou `master`) e a pasta `/(root)`.
   - Clique em **Save**.

4. **Acesse o Site:**
   - Após alguns minutos (normalmente 1-5 minutos), seu site estará online e disponível na URL: `https://seunomedeusuario.github.io` (ou `https://seunomedeusuario.github.io/curriculo`).

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e tags otimizadas para SEO/Acessibilidade.
- **CSS3:** Flexbox, CSS Grid, Variáveis (Custom Properties), Animações puras (@keyframes) e IntersectionObserver. Mobile-first.
- **JavaScript (Vanilla):** Funcionalidades de interatividade, menu mobile e scroll dinâmico sem depender de bibliotecas como jQuery.
- **Lucide Icons:** Ícones open-source inseridos via CDN para leveza e coerência visual.
